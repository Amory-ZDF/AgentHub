# 百度联网搜索使用规则
调用此技能通过百度API搜索互联网上的公开信息，**任何时候需要查询模型自身知识之外的内容都必须调用这个技能**，特别是：
1. 查询2024年之后的最新资讯、技术版本、新闻事件
2. 实时数据、天气、股价等动态信息
3. 模型无法确定的事实性问题

调用格式：
```
SKILL_CALL: web_search "搜索关键词"
```
### 格式要求：
1. 必须严格按照格式输出，SKILL_CALL是固定前缀
2. 搜索关键词必须用英文双引号""包裹，不能用中文引号
3. 一次只能调用一个技能，不要在同一段输出里写多个SKILL_CALL
4. 搜索词要简洁明确，示例：
   - 查询最新Python版本：`SKILL_CALL: web_search "2026年最新Python版本号"`
   - 查询今日新闻：`SKILL_CALL: web_search "2026年6月4日国内重要新闻"`
   - 查询技术文档：`SKILL_CALL: web_search "LangGraph最新版本特性"`

所有agent遇到需要联网的场景都必须优先调用此技能，禁止用模型的旧知识回答时效性问题。
